# MiniKlaim

> Run it. Klaim it.

A MiniPay Mini App where every hex you run through becomes yours on a shared city map. Built on Celo.

## Status

🚧 **Pre-MVP: design phase.** No code yet.

## Concept (one-liner per layer)

- **Mechanic:** H3 hex grid. Pass through a hex while running and it's yours.
- **Multiplayer:** Async. The map is shared and persistent. First to claim a hex owns it.
- **Web3:** Wallet auth via MiniPay. USDT season pass unlocks features. Achievement NFTs on Celo.
- **Stack:** Next.js 15 + Drizzle + PostGIS + MapLibre. Foundry contracts on Celo. All hosted on Railway.
